#  Learning

**Это мой личный полигон для экспериментов с кодом.**

Здесь я учусь, ошибаюсь, расту и превращаю сложное в работающие вещи.

---

##  Цель этого репозитория

Не быть идеальным. Быть честным.  
Каждый файл здесь — шаг вперёд.  
Каждая ошибка — урок.  
Каждый коммит — доказательство того, что я не стою на месте.

---

##  Что здесь можно найти

- `basics/` — повторение основ Python (циклы, функции, списки)
- `algorithms/` — задачи на поиск, сортировку, рекурсию
- `fastapi/` — маленькие API для тренировки
- `scraps/` — черновики и эксперименты

---

##  Принципы работы

1. **Пишу код каждый день.** Даже 5 строк — это прогресс.
2. **Не копирую без понимания.** Каждая строка должна быть осмыслена.
3. **Коммичу часто.** Каждый шаг — как закрепление в истории.
4. **Не боюсь ошибок.** Ошибка — это не конец, это диагностика.
